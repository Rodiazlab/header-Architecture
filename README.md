# ArchiFlex Landing Page

Landing page de arquitectura desarrollada como práctica de maquetación con HTML y CSS.

## Demo

https://rodiazlab.github.io/header-Architecture/

## Descripción

Proyecto centrado en la maquetación de una sección hero a partir de un diseño previo.

Se ha trabajado principalmente la estructura del layout, la jerarquía visual y el posicionamiento de elementos.

## Tecnologías

* HTML5
* CSS3
* CSS Grid
* Google Fonts (Poppins y Playfair Display)

## Objetivo

Reproducir un diseño UI y mejorar el control sobre maquetación, tipografía y composición visual.
